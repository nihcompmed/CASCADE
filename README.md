CASCADE: Chaotic Attractor Sensitivity for Cardiac Anomaly Detection
Overview

CASCADE is an online, personalized framework for cardiac arrhythmia detection that reframes the problem as dynamical regime transition detection rather than static classification. The method leverages entropy-tuned chaotic reservoir computing (DynML) to model short-term ECG dynamics and identify anomalies as failures of predictability.

Unlike conventional deep learning approaches that rely on large labeled datasets, CASCADE learns only from normal cardiac rhythms and detects arrhythmic events through deviations from learned patient-specific dynamics.
